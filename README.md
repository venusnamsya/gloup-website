# Gloup Beauty Website

## Overview

Gloup Beauty Website is a responsive beauty and salon booking website developed using HTML and CSS. The website allows users to explore beauty services, browse different salons and parlours, register or log in, and book appointments.

## Features

- Responsive homepage
- User registration and login pages
- Beauty service listings
- Parlour profiles
- Booking page
- Booking confirmation page
- Contact page
- Navigation between pages
- Attractive and modern user interface

## Technologies Used

- HTML5
- CSS3

## Project Structure

- `index.html` – Home page
- `login.html` – User login
- `register.html` – User registration
- `services.html` – Beauty services
- `parlours.html` – List of beauty parlours
- `booking.html` – Appointment booking
- `confirmation.html` – Booking confirmation
- `contact.html` – Contact page
- `style.css` – Website styling

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html`.
4. Run the project using Live Server or open the file in your web browser.
